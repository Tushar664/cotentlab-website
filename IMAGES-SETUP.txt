IMAGES SETUP — ek baar ka kaam
==================================================

STEP 1: Neeche wali har URL browser mein kholo aur
        image ko Right-click -> Save image as... karke
        SAME NAAM ke saath ek folder 'images' mein save karo.

STEP 2: GitHub repo kholo -> Add file -> Upload files ->
        poora 'images' folder drag karo -> Commit changes.
        (Auto-deploy se site pe khud aa jayengi)
